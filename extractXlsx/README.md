## Extract non-code lines from src file to Excel xlsx file

1. translated src files to `jpn` folder
2. original src files to `eng` folder
3. Run `extract.py`
4. excel output in `out` folder

Example output:  
`b01.src.xlsx`

| Line        | Jpn           | Eng  | Edit |
| ------------- |-------------| -----|---|
| 45    |「……ま、こんなもんか」 |「...Well, that should do it.」  ||
