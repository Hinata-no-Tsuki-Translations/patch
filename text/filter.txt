//ガンマ補正（便宜上フィルターとしています）
//立ち絵に自動でガンマ補正を行うための設定ファイルです

//①「#filter 任意の文字」コマンドでガンマ補正の使用を宣言（#「filter 夜」など）
//②以降、「#cg」コマンドで表示された立ち絵に指定した任意の文字に対応したガンマ補正が行われます
//③ガンマ補正を終了する際は「#filter default」コマンドを使用してください

//【name】	任意の名前　これが「#filter」コマンドの値として使用されます
//【object】	対象となるレイヤ（bg=背景のみ、cg=立ち絵全て、all=画面全体、cg0～＝個別指定もできますが、メモリ消費が増えるのでオマケ程度に）
//【rgamma】	赤成分のガンマ値 ( 0.0 ～ 1.0 ～ 9.0 ) を指定します
//【rfloor】	赤成分の出力最低値 ( 0 ～ 255 ) を指定します
//【rceil】	赤成分の出力最大値 ( 0 ～ 255 ) を指定します
//【ggamma】	緑成分のガンマ値 ( 0.0 ～ 1.0 ～ 9.0 ) を指定します
//【gfloor】	緑成分の出力最低値 ( 0 ～ 255 ) を指定します
//【gceil】	緑成分の出力最大値 ( 0 ～ 255 ) を指定します
//【bgamma】	青成分のガンマ値 ( 0.0 ～ 1.0 ～ 9.0 ) を指定します
//【bfloor】	青成分の出力最低値 ( 0 ～ 255 ) を指定します
//【bceil】	青成分の出力最大値 ( 0 ～ 255 ) を指定します
//【grayscale】	グレースケール（白黒化）するかどうか（true=する、false=しない）

//filter:	【name】	【object】	【rgamma】【rfloor】【rceil】	【ggamma】【gfloor】【gceil】	【bgamma】【bfloor】【bceil】	【grayscale】
filter:	default	all	1.0	0	255	1.0	0	255	1.0	0	255	false
filter:	夜	cg	0.8	0	200	0.8	0	200	1.2	0	200	false
filter:	夕	cg	2.0	0	255	1.0	0	255	1.0	0	255	false
filter:	白黒	cg	1.0	0	255	1.0	0	255	1.0	0	255	true
filter:	反転	all	1.0	255	0	1.0	255	0	1.0	255	0	false
filter:	黒	bg	1.0	0	128	1.0	0	128	1.0	0	128	false

filter:	黒1	cg1	1.0	0	128	1.0	0	128	1.0	0	128	true


//【prefix】	フィルターをかけたくない画像の判別文字（頭からn文字）
//例えば、【prefix】を「em」とした場合、「em～～～」という画像ファイルはフィルターの対象外になります
//集中線をはじめとしたエフェクト画像にはフィルターをかけたくない、といった場合などに使用してください
//※cgを対象に設定したときのみ、この設定は反映されます（allやbgを設定しているときは反映されません）
