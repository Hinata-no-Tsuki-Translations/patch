//フェイス関連付け
//#faceコマンドを使用する場合、ここで各種設定を行ってください

//■フェイスウィンドウの基本的なシステム
//①スクリプト中で「#face on」を記述し、フェイスウィンドウの使用を宣言します
//②スクリプト中の「【名前】」のところで、その名前に対応する立ち絵を自動的に拾ってきて表示します
//③任意に「#face ファイル名 x y」を指定すると、自動的な処理ではなく、任意指定の画像（と座標）を優先して表示します
//④フェイスウィンドウを表示したくないときは「#face off」で自由に非表示にできます

//自動表示は、実際に画面上に表示されている立ち絵に限ります
//また、フェイスウィンドウを使用する場合の立ち絵命名規則は、「制作にあたって.txt」にある命名規則に沿っている必要があります

//layout:全体的な設定です
//【prefix】フェイス画像の頭につける文字
//	立ち絵が「han_0a000」で【prefix】が「f_」だった場合、「f_han_0a000」という画像をフェイス画像として使用します
//	voidを指定した場合、立ち絵がそのままフェイス画像として使用されます
//	なお、立ち絵のサイズ指定は常に「0」が使用されます
//	例えば、「han_1a000」の立ち絵が表示されているときに取得されるフェイス画像は「han_0a000」になります
//	この辺りの処理は「TScript.tjs」の450行付近にあります
//【x】デフォルトのx座標
//	画像の中心のx座標となります（画像左端ではないので注意）
//【y】デフォルトのy座標
//	高さは画像上端で計算します

//layout:	【prefix】	【x】	【y】
layout:	f_	94	466

//chara:	【対応するキャラクター名】	【対応する画像ファイルの頭3文字】
chara:	【ヒナタ】	hin
chara:	【ナナシ】	hin
chara:	【エルフ】	hin
chara:	【イバラ】	iba
chara:	【コノミ】	kon
chara:	【ツキヨ】	tuk
